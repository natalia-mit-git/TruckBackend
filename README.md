# TruckBackend

![Tests](https://github.com/natalia-mit-git/TruckBackend/actions/workflows/dotnet-tests.yml/badge.svg)

Simple ASP.NET backend for truck load management.

## Features

- Create truck loads
- Calculate total truck weight
- Service layer architecture
- Unit tests with xUnit

## Tech Stack

- ASP.NET Core
- C#
- xUnit
- REST API

## Endpoints

API documentation and testing available via Swagger UI when running the project.

## Run the project

dotnet build  
dotnet run  

## Run tests

dotnet test
