# TruckBackend API

Small REST API built with ASP.NET Core for managing truck loads.

## Features

- Create truck loads
- Calculate total truck weight
- Service layer architecture
- Unit tests with xUnit

## Tech Stack

- ASP.NET Core
- C#
- xUnit
- REST API

## Endpoints

API documentation and testing available via Swagger UI when running the project.

## Run the project

dotnet build  
dotnet run  

## Run tests

dotnet test
